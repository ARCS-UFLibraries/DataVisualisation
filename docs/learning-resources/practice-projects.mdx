Projects allow you to apply concepts learned throughout this guide.

Rather than following tutorials, you'll solve real-world problems using data.

## Project Workflow

A typical data analysis project includes:

1. Define the problem
2. Collect the data
3. Clean the data
4. Explore the data
5. Analyze the data
6. Visualize findings
7. Communicate results

---

## Portfolio Tip

As you complete projects, publish them on GitHub.

A portfolio containing well-documented projects demonstrates your technical skills and analytical thinking to potential employers and graduate programs.
