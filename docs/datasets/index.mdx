# Datasets

The datasets in this guide are used throughout the lessons, exercises, and case studies.

Each dataset includes:

- A description of the data
- Learning objectives
- Download links
- Associated lessons
- Google Colab notebooks (where available)

Working with real-world datasets helps develop practical data analysis skills and reinforces the concepts introduced in each module.
