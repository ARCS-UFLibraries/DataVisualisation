# Working with Data

Before you can create charts or build machine learning models, you need to understand your data.

In this module, you will learn how to work with datasets using **Pandas**, a Python library that helps you read, explore, clean, and organize data. Step by step, you will learn how to move from a raw file to useful insights.

### Why this module matters

Real datasets are often messy. They may contain missing values, extra columns, or information that needs to be sorted and cleaned before you can analyze it.

This module helps you build the skills you need to work with data confidently. These are the same core steps analysts use before creating dashboards, reports, or models.

### What you’ll learn

In this module, you will learn how to:

- Read CSV and Excel files
- Understand DataFrames
- Explore a dataset
- Select rows and columns
- Filter and sort data
- Clean and prepare data
- Group and summarize information
- Export your results

### How you’ll learn

You will follow a simple workflow throughout the module:

1. Load the data
2. Explore the dataset
3. Clean and prepare the data
4. Analyze the data
5. Summarize findings
6. Export results

By following the same workflow again and again, you will start to see how each Pandas skill fits into a real data analysis task.

### Dataset

Most examples in this module use a **Students Performance** dataset. This gives you a familiar context while you practice reading files, inspecting data, filtering rows, and summarizing results.

### Final case study

At the end of the module, you will complete a case study that brings everything together.

You will work through a full data analysis task from start to finish and apply the skills you learned in the lessons before it.
